/* EXTERNAL LIBRARIES USED */
dpkt

/* HOW TO RUN PROGRAM */
1) Install dpkt on device, using pip using the command "pip install dpkt" in the Command Prompt.
2) Open Command Prompt and change directory to where you extracted the .zip file, where the analysis_pcap_arp.py is located
and this file is located.
3) Run the program using the command line: "python analysis_pcap_arp.py"
4) Input the assignment2.pcap file by typing in: "assignment3_my_arp.pcap" to the input prompt (no quotation marks)

NOTE: Make sure assignment3_my_arp.pcap is in the same directory as analysis_pcap_arp.py or the program will not be able to find
the file.
